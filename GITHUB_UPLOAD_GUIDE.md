# 📤 GitHub Upload Guide
**Created by D**

## Method 1: GitHub Desktop (Easiest)

1. **Download GitHub Desktop**: https://desktop.github.com/
2. **Install GitHub Desktop**
3. **Open GitHub Desktop**
4. **File** → **Add Local Repository**
5. **Choose folder**: `C:\Users\user\CascadeProjects\anticheat`
6. **Commit changes**: "Initial commit: FiveM AntiCheat System"
7. **Publish repository**: Click "Publish Repository"
8. **Name**: `fivem-anticheat`
9. **Public**: Yes
10. **Publish Repository** ✅

## Method 2: GitHub Website (Manual Upload)

1. **Go to**: https://github.com
2. **Sign in** to your account
3. **Click**: `+` → **New repository**
4. **Repository name**: `fivem-anticheat`
5. **Description**: `FiveM AntiCheat System by D`
6. **Public**: ✅
7. **Click**: **Create repository**
8. **Click**: **uploading an existing file**
9. **Drag and drop** all files from the anticheat folder:
   - client.lua
   - server.lua
   - config.lua
   - fxmanifest.lua
   - README.md
   - QUICK_START.md
   - .gitignore
10. **Commit changes**: "Initial commit"
11. **Commit new files** ✅

## Method 3: Install Git (Advanced)

1. **Download Git**: https://git-scm.com/download/win
2. **Install Git** (default settings)
3. **Restart** your computer
4. **Double-click**: `publish_to_github.bat`

## 🎯 Recommended: Method 1 (GitHub Desktop)

**Why?**
- ✅ Easiest for beginners
- ✅ Visual interface
- ✅ No command line needed
- ✅ One-click publishing
